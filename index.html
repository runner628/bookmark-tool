    📲 如何在手机上导入书签？
    方法一：下载 HTML 后传到电脑，在 Chrome/Edge/Firefox 中导入
    方法二：通过微信「文件传输助手」传到电脑再导入
    方法三：用 AirDrop（苹果）或 Nearby Share（安卓）传到电脑

    电脑端导入步骤：
      Chrome/Edge → 书签管理器 → 导入 → 选择 HTML 文件
      Firefox → 管理书签 → 导入和备份 → 从 HTML 导入
